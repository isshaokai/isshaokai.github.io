## 💰 核心补贴政策

### 1. Ecobonus（生态奖金）

**政策概述：**

| 项目 | 内容 |
|------|------|
| **实施时间** | 2007 年至今 |
| **适用范围** | 建筑能效改造 |
| **热泵补贴** | 50%-65% 抵扣 |
| **抵扣方式** | 10 年个人所得税抵扣 |

**补贴比例：**

| 改造类型 | 补贴比例 | 上限金额 |
|----------|----------|----------|
| 基础能效改造 | 50% | €96,000 |
| 高效热泵系统 | 65% | €30,000 |
| 热泵 + 光伏 | 65% | €48,000 |
| 热泵 + 储能 | 65% | €60,000 |

**申请条件：**
- ✅ 既有建筑改造（新房不适用）
- ✅ 能效提升至少 20%
- ✅ 使用认证设备
- ✅ 专业安装资质

---

### 2. Superbonus（超级奖金）

**政策概述：**

| 项目 | 内容 |
|------|------|
| **实施时间** | 2020-2024（已逐步退坡） |
| **补贴比例** | 最高 110% |
| **现状** | 2024 年起降至 70%，2025 年 65% |
| **适用** | 主要住宅 + 小型公寓 |

**超级奖金 + 热泵：**

```
2023 年：110% 全额补贴
    ↓
2024 年：70% 补贴
    ↓
2025 年：65% 补贴
    ↓
2026 年：预计 50% 补贴
```

**注意事项：**
- ⚠️ 政策持续退坡
- ⚠️ 审批流程复杂
- ⚠️ 需要整体改造（非单一设备）

---

### 3. Conto Termico 2.0（热能账户）

**政策概述：**

| 项目 | 内容 |
|------|------|
| **实施机构** | GSE（能源服务公司） |
| **补贴方式** | 直接拨款（非税收抵扣） |
| **发放周期** | 2-5 年分期 |
| **适用对象** | 公共部门 + 私营企业 + 个人 |

**补贴标准：**

| 设备类型 | 补贴额度 | 发放周期 |
|----------|----------|----------|
| 空气源热泵 | €300-500/kW | 2 年 |
| 地源热泵 | €500-800/kW | 5 年 |
| 热泵 + 光伏 | 额外 +20% | 5 年 |
| 智能控制系统 | 额外 +10% | 2 年 |

**优势：**
- ✅ 直接拨款，无需纳税
- ✅ 审批相对简单
- ✅ 可与 Ecobonus 叠加

---

### 4. 光伏 + 储能专项政策

#### 4.1 Bonus Ristrutturazioni（翻新奖金）

| 项目 | 内容 |
|------|------|
| **补贴比例** | 50% |
| **上限金额** | €96,000 |
| **适用范围** | 建筑翻新 + 光伏安装 |
| **抵扣方式** | 10 年税收抵扣 |

#### 4.2 净计量政策（Scambio sul Posto）

**政策要点：**

```
┌─────────────────────────────────────────────────────────┐
│                   净计量政策要点                        │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   发电 → 自用优先                                       │
│    ↓                                                    │
│   余电 → 上网 → 获得补偿                                │
│    ↓                                                    │
│   不足 → 电网购电                                       │
│    ↓                                                    │
│   年度结算 → 多退少补                                   │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**补偿标准：**
- 上网电价：约 €0.08-0.12/kWh
- 年度结算：可结转下一年
- 税务处理：个人免税，企业征税

#### 4.3 储能专项支持

| 政策 | 内容 |
|------|------|
| **补贴比例** | 50%（与 Ecobonus 一致） |
| **上限金额** | €60,000 |
| **技术要示** | 锂电为主，需认证 |
| **并网要求** | 智能电表 + 远程监控 |

## 🔥 热效能支持政策

### 1. 最低能效标准（MEPS）

**欧盟法规：**

| 设备类型 | 最低能效 | 实施时间 |
|----------|----------|----------|
| 空气源热泵 | SCOP ≥ 3.5 | 2025 年 |
| 地源热泵 | SCOP ≥ 4.0 | 2025 年 |
| 高温热泵 | SCOP ≥ 3.0 | 2026 年 |

**意大利额外要求：**
- 必须配备智能控制系统
- 必须支持远程监控
- 必须与可再生能源集成

### 2. 能效认证体系

**认证等级：**

```
A+++  →  最高能效（推荐，补贴最高）
A++   →  高能效
A+    →  良好能效
A     →  标准能效（最低要求）
B     →  低于标准（淘汰中）
C/D   →  禁止销售
```

**补贴差异：**

| 能效等级 | 补贴系数 | 示例（€10,000 设备） |
|----------|----------|----------------------|
| A+++ | 1.0 | €6,500 |
| A++ | 0.9 | €5,850 |
| A+ | 0.8 | €5,200 |
| A | 0.7 | €4,550 |

### 3. 区域供热支持

**政策要点：**

| 项目 | 内容 |
|------|------|
| **适用范围** | 城市区域供热系统 |
| **补贴比例** | 40-50% |
| **技术路线** | 大型热泵 + 余热回收 |
| **资金来源** | 欧盟复苏基金 + 国家配套 |

## 📋 政策申请要点

### 关键文件清单

- [ ] 技术方案（含能效计算）
- [ ] 设备认证证书
- [ ] 安装资质证明
- [ ] 建筑所有权证明

### 常见拒批原因

| 原因 | 占比 | 预防措施 |

## 🎯 总结与建议

### 核心结论

1. **市场机会大** — 意大利热泵市场处于高速增长期
2. **政策支持强** — 多重补贴政策叠加
3. **一体化趋势** — 光储 + 热泵是主流方向
4. **时效性强** — 补贴政策持续退坡，宜尽早布局

### 行动建议

| 优先级 | 行动 | 时间 |
|--------|------|------|
| 🔴 高 | 市场调研 + 伙伴对接 | 1 个月内 |
| 🔴 高 | 产品认证准备 | 3 个月内 |
| 🟠 中 | 试点项目落地 | 6 个月内 |
| 🟠 中 | 本地团队建设 | 6 个月内 |
| 🟡 低 | 规模化扩张 | 12 个月内 |

### 风险提示

- ⚠️ 政策持续变化，需密切关注
- ⚠️ 本地化要求高，需深度本地化
- ⚠️ 竞争激烈，需差异化定位
- ⚠️ 回款周期长，需资金准备
